# Quantitative Strategies - Thomas Heckel

Collection of Projects to Test Various Quantitative Strategies / Approaches

## Structure
- `src/`: Python modules
- `src/quantstrategies/vcv/` : vcv estimations
- `src/quantstrategies/factor_regressions/` : factor regressions
- `examples/` : simple scripts to launch demos
- `tests/` : unitary tests
